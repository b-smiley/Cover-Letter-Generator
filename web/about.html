Cover Letter Generator

Overview

This is a simple cover letter generator that allows you to input your information and generate a cover letter. The cover letter is generated in a PDF document format.

Goal of this application

The goal of this application was to find a way to automate the process of creating cover letters. This application is designed to be simple and easy to use.

How to Use

Create template file that has placeholder values for you to fill out. The template file name should not be changed from "cover_letter_template.txt". For a detail guide on each functionality please refer to the Help Page.

Features

  • Create a simple custom cover letter template
  • Fill out your template information into a form that automatically populates your cover letter.
  • Save the cover letter to a file destination in .pdf format
  • Spellcheck warnings to ensure that your cover letter is free of spelling errors. (Still recommended to proofread)
  • Use special variables to perform additional automation
    • Your_Name && Company_Name will be used to name the PDF file (if not specified, the file will be named "cover_letter.pdf")
    • Todays_Date will be used to populate the date in the cover letter
